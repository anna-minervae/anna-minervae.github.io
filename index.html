FireWatch - Spring 2023

FireWatch was an application I created for my Environmental Science class in my junior year of high school. I designed a bushfire simulation based on various scientific studies. Made in C# with the Unity game engine, FireWatch works on a tile system. Each tile has both a vegetation property and an elevation property. By analyzing each tile's properties, the algorithm creates a visual prediction of the bushfire's behavior. I was inspired by my Australian heritage, and I hope to refine this one day.
